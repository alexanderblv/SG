# 📝 Git Commands для деплоя

## Команды для пуша изменений:

```bash
# Добавить все файлы
git add .

# Создать коммит
git commit -m "Migrate to React SDK v2.13.0 - fix CDN issues"

# Запушить на GitHub
git push origin main
```
